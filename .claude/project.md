# Project Overview

OrgConnect is a centralized web platform exclusively for the **Holy Angel University - School of Computing**.

Its purpose is to help students discover all official School of Computing student organizations from a single platform.

Currently, students rely on Facebook pages, posters, classmates, and word-of-mouth to learn about organizations.

OrgConnect solves this problem by providing a modern organization discovery experience.

This project is **NOT** a social media platform.

This project is **NOT** an organization management system.

This project is **NOT** a membership registration platform.

The primary goal is to help students understand each organization before deciding which one best matches their interests.

---

# Primary Goals

The website should allow students to easily discover:

- What organizations exist
- What each organization offers
- What skills they can learn
- What events each organization conducts
- What opportunities are available
- Which organization best matches their interests

---

# Core Features

## Homepage

The homepage serves as the central hub.

It showcases every official School of Computing organization.

Each organization is displayed as a modern interactive card.

Each card contains:

- Organization logo
- Organization name
- Short description
- Skills offered
- Learn More button

---

## Organization Details Page

Clicking an organization opens a dedicated page.

Each organization page contains:

- Organization overview
- Mission and vision
- What the organization offers
- Skills students can learn
- Activities
- Workshops
- Competitions
- Events
- Opportunities
- Officer information (if available)
- Gallery
- Social media links
- Contact information

---

## AI Recommendation Assistant

The highlight feature of OrgConnect is the AI Recommendation Assistant.

Students can describe:

- their interests
- skills
- hobbies
- career goals
- preferred activities

The AI recommends one or more organizations that best fit the student.

The AI should also explain **why** each recommendation was made.

Example:

> Based on your interest in web development, UI/UX, and hackathons, the Google Developer Student Club is an excellent choice because it regularly conducts workshops, coding events, and collaborative projects.

The AI acts as a recommendation assistant.

It does not replace organization information.

---

## Search

Students can search organizations by:

- Name
- Skill
- Category
- Technology
- Interest

---

## Events

Students can browse upcoming events hosted by organizations.

Each event contains:

- Title
- Date
- Venue
- Description
- Hosting organization

# What This Project Does NOT Include

Do NOT implement:

- User registration
- Login
- Sign up
- Authentication
- Student dashboard
- Organization dashboard
- Messaging
- Chat between users
- Posts
- Likes
- Comments
- Following organizations
- Membership applications
- Admin management pages (unless explicitly requested)

This platform is focused on **organization discovery and AI-assisted recommendations**.
